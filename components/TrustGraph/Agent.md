# Agent

**Role:** Constitutional AI
**Type:** Governance
**Status:** Online

The constitutional AI agent framework. Every agent operates bounded by SOUL.md and CONSTITUTION.md. Agents receive **commands** from [[Navigator]] and **report** scores to [[TrustGraph]]. Each agent holds a [[CAC]] (Compute Access Certificate) — a prepaid compute credential that gates inference access.

## Agent Principles

1. **Constitutional bounds** — all actions must pass constitutional review
2. **Escalate uncertainty** — never confabulate, escalate instead
3. **Asymmetric trust** — slow to earn, fast to lose
4. **Full auditability** — all actions logged and reversible where possible
